# Online Ticket Reservation System

## Overview
The Online Ticket Reservation System is a mini project developed using C++ with Object Oriented Programming (OOP) and File Handling concepts. The system allows users to book, search, update, view, and cancel tickets efficiently through a menu-driven interface.

Reservation details are stored permanently using file handling, making the project suitable for demonstrating real-world implementation of OOP concepts in C++.

---

## Features
- Book New Tickets
- View All Reserved Tickets
- Search Ticket Details
- Update Ticket Information
- Cancel Tickets
- Permanent Data Storage using File Handling
- Ticket Receipt Generation
- Automatic Discount Calculation
- Exception Handling for Invalid Inputs
